# Study Up — Application de Gestion des Tâches Étudiantes

Application desktop Java développée dans le cadre du PFE DUT 2024.
Conçue pour aider les étudiants à organiser leurs tâches et améliorer leur productivité.

## Fonctionnalités
- Création et gestion de projets et tâches
- Timer Pomodoro (25 min travail / 5 min pause)
- Calendrier intégré
- Tableau blanc (Whiteboard)
- Statistiques de progression par jour
- Système de notifications
- Authentification (inscription / connexion)

## Stack technique
- Java, JavaFX, SceneBuilder
- MySQL, XAMPP, PhpMyAdmin
- Eclipse IDE
- Astah UML

## Lancer le projet
1. Installer XAMPP et démarrer MySQL
2. Importer la base de données (fichier .sql dans /database)
3. Ouvrir le projet dans Eclipse
4. Lancer Main.java
